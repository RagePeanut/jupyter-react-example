# Jupyter React Example

```bash
# Setup your development environment.
conda deactivate && \
  make env-rm # If you want to reset your env.
make env && \
  conda activate jupyter-react-example
```

```bash
make install
```

```bash
# Start the frontend and backend servers.
echo open http://localhost:8080
make start
```
